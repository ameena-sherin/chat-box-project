# 💬 Chat Bot Project

## 📌 Description
A simple chat bot built using HTML, CSS, and JavaScript. The bot responds to user inputs using predefined logic without AI.

---

## 🚀 Features
- User chat input
- Bot replies using simple logic
- Interactive UI design
- Beginner-friendly project
